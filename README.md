# Auwt Aelse - Bistro Website

Officiële website van Bistro Auwt Aelse in Elsloo.

## Technische Specificaties

- **Framework:** Next.js 16.1.6
- **UI Library:** React 19.2.4
- **Styling:** CSS Modules
- **TypeScript:** 5.9.3
- **Deployment:** Cloudflare Pages

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Deployment

De website is geconfigureerd voor static export naar Cloudflare Pages.

**Build Command:** `npm run build`  
**Output Directory:** `out`

## Features

- Premium zwart/wit design
- Fully responsive (mobile-first)
- Tweetalig (NL/EN)
- SEO geoptimaliseerd
- AVG/GDPR compliant
- Performance geoptimaliseerd
- Toegankelijk (WCAG 2.1)
